🚽 Smart Sanitation & Access Locator

🌐 Live Demo: https://smart-sanitation.vercel.app/

An AI-enabled smart sanitation platform that helps monitor public toilet cleanliness and guides users to nearby hygienic facilities in real time. The system aims to improve urban hygiene, accessibility, and citizen awareness.

📌 Problem Statement

Public sanitation facilities often suffer from:

❌ Poor cleanliness monitoring

❌ Lack of real-time status updates

❌ Difficulty finding nearby usable toilets

❌ Water availability issues

❌ No centralized reporting system

This project provides a smart, AI-driven solution to make cities cleaner and safer.

🚀 Features

🧠 AI Cleanliness Detection
Detects litter, wet floors, and overflow conditions.

📊 Live Cleanliness Grade
Displays real-time hygiene score outside facilities.

📍 Nearby Toilet Locator
Guides users to alternative toilets within 1–2 km.

💧 Water Availability Monitoring
Tracks water status in facilities.

📱 User-Friendly Dashboard
Simple and responsive UI for quick access.

⚡ Real-Time Updates
Instant status visibility for users and authorities.

🧩 How It Works

Sensors / AI monitor toilet conditions

System calculates cleanliness score

Data is sent to the cloud (Supabase)

Users view live status on the web app

If a toilet is खराब, users get nearby alternatives
